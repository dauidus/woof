This plugin integrates WordPress Customizer with Google Web Fonts, to add and use google fonts to any themes, no coding needed.

== Description ==
Google Web Fonts Customizer ( GWFC ) will manage Google Web Fonts to be used on your site using WordPress Customizer live preview, with wide variety usage from body tags until list tags.
You can change your font weight, font style and also font color.

= What does this plugin do? =
* You can use this plugin on **any WordPress theme's**.
* With this plugin, you can using **Google Web Fonts** into your WordPress theme's typography.
* This plugin integrate with **WordPress Customizer live preview**, 
* That mean you can **preview your typography change in realtime** before you save it.
* You can change your **font weight**, **font style** and also your **font color**.
* This plugin **automatically** change your fonts on theme's typography, no need to edit theme's stylesheet.

**Please note:** I am currently working on new feature for this plugin, any requests and suggestions can be sent to the following email **chanif.alfath@gmail.com**. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Install Google Web Fonts Customizer ( GWFC ) either via the WordPress.org plugin directory, or by uploading the files to your server
2. After activating Google Web Fonts Customizer ( GWFC ) by WordPress.com, you will get new menu ( Fonts Customizer ) in Administration Panel.
3. Click the Fonts Customizer menu button, you will get new section ( Fonts Customizer ) on WordPress Customizer live preview.
4. Activate Google fonts on tags / area by enabled / disabled tags selection.
5. After enable tags, select font type, weight, style and color of Google Web Fonts that you wanna use for the tags / area.
6. Do not forget to save before you leave WordPress Customizer live preview.
7. That's it. You're ready to go!
